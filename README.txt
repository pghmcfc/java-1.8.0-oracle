Get the JDK as follows:

* Go to http://www.oracle.com/technetwork/java/javase/downloads/index.html
* Click on the JDK Download button for Java SE 8u11
* You need to accept the Oracle Binary Code License Agreement for Java SE before continuing
* Download the Linux x86 - Compressed Binary (jdk-8u11-linux-i586.tar.gz),
  or on x86_64, the Linux x64 - Compressed Binary (jdk-8u11-linux-x64.tar.gz)

Get the timezone updater as follows:

* Go to http://www.oracle.com/technetwork/java/javase/downloads/index.html
* Click on the Download button for JDK Timezone Updater Tool
* Check the box agreeing to the Java SE Timezone Updater License Agreement
* Download the Java SE Timezone Updater 1.4.5 (tzupdater-1_4_5-2014e.zip)
